### 🚀 项目介绍

欢迎来到 **ChatWithEveryone** —— 你的专属 AI 聊天伴侣！🎉

这是一款基于 SwiftUI 精心打造的 macOS 桌面聊天应用，让你随时随地和顶尖 AI 模型畅快对话。无论你是想调用 **DeepSeek** 的深度推理能力，接入 **硅基流动 (SiliconFlow)** 的高性能大模型，还是连接你自己的 **自定义 API 端点**，这里统统给你准备好了，开箱即用！

### 💻 适用平台

专为 **macOS** 而生，完美适配 Mac 桌面体验。

### 📦 安装方式

简单到只需一步 —— 把 App 直接拖进你的「应用程序」文件夹，立刻就能嗨聊起来！✨
